Présentation

Avec Wp Easy RSS, il n'a jamais été aussi simple d'avoir un ou plusieurs lecteurs de flux RSS dans votre projet web Wordpress de façon dynamique !!!
En effet, il vous propose, avec un look simple et épuré, de montrer vos flux de manière dynamique avec son scrolling automatique vertical.

Installation

2 méthodes :

  • Installation manuel
  • Installation Wordpress

Installation Manuel

  1. Sur Github, allez sur le répertoire "projetPluginWp"
  2. En bas à droite du répertoire, vous trouvez un bouton "Download Zip", cliquez dessus
  3. Votre téléchargement terminé, dézippez le zip "projetPluginWp"
  4. Dans le nouveau dossier copier le dossier "Wp_easy_RSS"
  5. Maintenant allez dans le dossier de votre projet Wordpress
  6. Dans le dossier de votre projet Wordpress, cliquez sur Wp-content
  7. Dans Wp-content, vous avez un dossier Plugins, cliquez dessus
  8. Collez le dossier "Wp_easy_RSS"
  9. Dans le tableau de bord Wordpress, cliquez sur "Extensions" et activez Wp_easy_RSS
  10. Vous avez installé le plugin

Installation Wordpress

  1. Sur Github, allez sur le répertoire "projetPluginWp"
  2. En bas à droite du répertoire, vous trouvez un bouton "Download Zip", cliquez dessus
  3. Votre téléchargement terminé, dézippez le zip "projetPluginWp"
  4. Allez dans le fichier projetPluginWp
  5. Zippez le dossier Wp_easy_RSS
  6. Dans le tableau de bord Wordpress, cliquez sur "Extensions"
  7. En haut, cliquez sur le bouton "ajouter"
  8. En haut, cliquez sur le bouton "mettre une extension en ligne"
  9. Maintenant, selectionnez Wp_easy_RSS.zip et cliquez "installer maintenant"
  10. L'Installation terminé, cliquez "activer l'extension"
  11. Vous avez intallé le plugin

Comment utiliser le plugin

Dans le tableau de bord Wordpress, cliquez sur "Apparence"
Maintenant, cliquez sur "Widgets"
Prenez et déposez le widget "Widget flux RSS" dans votre sidebar
Entrez votre titre
Entrez votre URL RSS
Cliquez sur Enregistrer
Maintenant sur votre page web vous avez le plugin "Wp_easy_RSS"

Customiser Wp_easy_RSS

Il est facile de customiser le widget "Wp_easy_RSS", regardez :

  • Allez dans le dossier de votre /projet Wordpress > wp-content > plugins > Wp_easy_RSS

Dossier du Widget

-Ici vous avez deux dossier, un pour le JQuery et l'autre pour le CSS

Dans le dossier jQuery

-Si vous voulez custom le JQuery, allez dans le dossier "JQuery" (Attention le système du scroll auto est là)

Dans le dossier style

-Si vous voulez custom le CSS, allez dans le dossier "style"
